Правила оформления исходного кода:

1. Именование Классов:
    UpperCamelCase { ExampleClass }

2. Именование переменных/данных(поле) класса
    lowerCamelCase { exampleVariable }

3. Именование свойств класса:
    UpperCamelCase

4. Именование функций/методов класса:
    UpperCamelCase

5. Расстановка блоков
    Стиль Олмана:
    if (<..>)
    {
        <..>
    }

6. Оформление арифмитических выражений:
    Дефолтный через пробел
    { i = 0; i < n; i++ }

7. Оформление комментариев:
(комментарии перед функциями)
    /*
        @return void
        kto? - kogda?
        chto eto?
    */
